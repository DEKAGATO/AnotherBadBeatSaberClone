Class SpectrumPlotter

A sort of plotting component for a test scene, that is not needed anymore. That scene was needed to visualize the audio analysis when developing it. Basically what this does is to spawn a bunch of small cubes from the generated audio data and visualize the peaks and flux/threshold levels while the song is playing.
